# www.simonplant.com

Personal website for Simon Plant.

## Tech Stack

- [Astro](https://astro.build) 6 (static)
- Tailwind CSS v4
- TypeScript (strict)
- PostHog analytics (consent-gated)

## Development

```sh
npm install
npm run dev       # localhost:4321
npm run build     # type-check + build
npm run preview   # preview build
```

## Environment Variables

Copy `.env.example` to `.env` and fill in your PostHog credentials.
