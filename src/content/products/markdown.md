## What it is

Markdown is a free, open-source markdown editor for iOS. Open any `.md` file straight from the Files app — iCloud Drive, Dropbox, or any other document provider. No vault. No account. No subscription.

## Stack

Rust core (tree-sitter-markdown parsing and formatting), native Swift frontend on TextKit 2. Apache 2.0 end to end — not open-core, not source-available.

## Status

**Pre-release.** The app runs on device and in the iOS Simulator with wikilinks, footnotes, and GFM tables working; it has not yet shipped to TestFlight or the App Store.

Built by autonomous agents using AIShore for sprint orchestration.
