## What it is

Markdown is a free markdown editor for macOS, Linux, Windows, Web, iOS, and Android. Open any `.md` file from a local folder or a cloud drive — iCloud Drive, Dropbox, Google Drive, OneDrive. No vault. No account. No subscription.

## Stack

Rust core. CodeMirror 6 editor. Tauri shells for desktop and mobile. Apache 2.0 end to end — not open-core, not source-available.

## Status

**Pre-M0.** There is no code yet — only docs and a backlog. The first milestone is a walking skeleton: open a file, edit it, save it back, measure the baseline. Everything else decorates that.

Built by autonomous agents using AIShore for sprint orchestration.
